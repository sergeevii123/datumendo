## Datumendo
App for migration of data from Filecoin/Ipfs and Arweave to Greenfield.

### Prerequsiites
User must have a wallet connected to the Greenfield testnet and have it's native token.

### Install
```
pnpm install
```

### start app in dev
```
npx next dev
```

### start app in prd
```
npx next build
npx next start
```
